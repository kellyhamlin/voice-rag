# Corpus

All documents here are **synthetic or public**. None contain real operator
data, credentials, hostnames, or IP addresses. Synthetic runbooks are marked
`type: synthetic-runbook` in their front matter.

To add public specs (optional): download RFC 3261 (SIP), RFC 3550 (RTP), and
RFC 4566 (SDP) as text into this folder. They are public-domain IETF documents.
